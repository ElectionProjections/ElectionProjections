About the thresholds

Solo parties need 5% of the national vote to win any seats; coalitions need 8%. Votes for anyone below threshold are redistributed proportionally among parties that clear it — this is why a poll showing a party at 3–4% can still mean zero seats.

Method. This tool uses the actual 2023 results in all 41 Sejm constituencies (source: PKW) and applies uniform national swing — each party's vote share moves by the same number of percentage points in every constituency as it does nationally. Real D'Hondt is then run separately within each constituency using its actual seat count (7–20), and the 41 results are summed. Whether a party clears the 5%/8% threshold is decided by its national vote share, exactly as in the real system. With this method the 2023 preset reproduces the actual outcome exactly: PiS 194 / KO 157 / Third Way 65 / Lewica 26 / Konfederacja 18.

Third Way, Lewica, and Confederation ran as single joint lists in 2023, so Polska 2050/PSL, Lewica/Razem, and Konfederacja/KKP are each split from their joint list's district-level vote using the national ratio between their named sub-affiliates. This assumes each pair had the same internal split in every constituency, which won't be exactly true — it's the most reasonable estimate available without separate historical lists. Uniform swing is also a simplification: real shifts vary by region, so a close result can tip either way in practice.

The map shows official constituency boundaries, dissolved from Poland's 380 powiats (counties) exactly as defined in the Election Code. A constituency's colour reflects whichever party wins the most seats there, not necessarily the most votes — in a multi-member district these usually match but can occasionally diverge.

Sources. 2023 results by constituency: Polish State Electoral Commission (PKW). Constituency boundaries: Polish Election Code Annex 1, geometry via powiat-level GIS data (Geoportal.gov.pl, public domain).
